About the Software

Our open-source software is a Python-based simulator designed to model voltage-controlled memristive switching in oxide-based devices. The software allows users to explore how device geometry, material properties, voltage conditions, and simulation parameters affect switching behavior.

The program is designed for research and education. It provides an interactive way to visualize oxygen vacancy migration, conductive filament formation, temperature evolution, electric potential, and current-voltage response.

View Software on GitHub

Main Features

  • Interactive Python graphical user interface
  • Simulation of VCM electroforming behavior
  • Visualization of oxygen vacancy concentration
  • Temperature and electric potential maps
  • Current-voltage curve generation
  • Adjustable oxide material properties
  • Adjustable geometry, voltage, and simulation parameters
  • Open-source code for research and educational use

Physical Processes Included

The simulator focuses on the coupled physical mechanisms involved in oxide-based resistive switching. These include electric-field-driven oxygen vacancy migration, Joule heating, conductivity changes, and conductive filament development.

By changing parameters in the software, users can observe how different material and device conditions influence the formation of conductive pathways and the resulting current-voltage behavior.

How to Run

After downloading the software from GitHub, open Terminal and navigate to the project folder. Then run: 

python3 VCMModelFinal1.py

Required Python packages may include NumPy, Matplotlib, SciPy, and Pillow. They can be installed with: 

python3 -m pip install numpy matplotlib scipy pillow

Purpose

The goal of this software is to make memristor modeling more accessible. Instead of requiring every user to build a full simulation from scratch, the program provides a starting point for visualizing and modifying VCM electroforming behavior.
