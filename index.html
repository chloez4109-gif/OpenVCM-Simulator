Research

Our research group studies resistive switching devices, memristor modeling, and computational methods for understanding materials used in neuromorphic computing systems.

ECM Memristor Modeling

We investigate electrochemical metallization memory devices and model the physical processes that control switching behavior.

VCM Electroforming Simulation

We study valence change memory devices, including oxygen vacancy migration, conductive filament formation, Joule heating, and current-voltage behavior.

COMSOL Multiphysics

We use multiphysics simulations to examine coupled electrical, thermal, and ionic transport processes in oxide-based memristors.

Open-Source Software Development

We develop accessible simulation software so students and researchers can visualize memristor behavior and explore device parameters interactively.

Open-Source Software

Our open-source software is a Python-based simulator for studying voltage-controlled memristive switching in oxide-based devices. The program allows users to adjust material properties, device geometry, voltage conditions, and simulation parameters to observe changes in oxygen vacancy concentration, temperature, electric potential, and current-voltage response.

Software Features

  • Interactive simulation of VCM electroforming behavior
  • Visualization of oxygen vacancy migration and conductive filament formation
  • Temperature, electric potential, and current-voltage plots
  • Adjustable oxide material and device parameters
  • Open-source Python code for research and education
