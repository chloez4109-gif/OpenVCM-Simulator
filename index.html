About Our Research

Our research focuses on resistive switching in metal-oxide devices, especially the formation and rupture of conductive filaments through oxygen vacancy migration, Joule heating, and electric-field-driven transport.

About the Software

This open-source Python software provides an interactive graphical interface for simulating voltage-controlled memristive behavior in oxide materials such as HfO₂, TaOₓ, TiOₓ, and ZrO₂. The program allows users to adjust device geometry, oxide properties, thermal parameters, and voltage sweeps to observe changes in current, temperature, electric potential, and oxygen vacancy concentration.

Key Features

  • Interactive Python GUI
  • 2D visualization of oxygen vacancy concentration
  • Temperature and electric potential maps
  • I-V curve plotting
  • Material selection for different oxide layers
  • Open-source and customizable code
